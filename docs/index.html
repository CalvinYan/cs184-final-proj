CS 184: Computer Graphics and Imaging, Spring 2023

Final Project: Augmented Reality Rendering

Yuntian Ke, Calvin Yan, Ruichi Zhang, Wentao Zhang

Summary

Our project focus on demonstrating some 2-D or 3-D renderings by augmented reality. We use the QR code as a mark so that we can locate the area and plane to render models on, then we can add some cool special effect(dynamic lighting, live model and animation) on the real world through computer vision and computer graphics.

Goals and Deliverables

What we plan to deliver?

Making a vedio demo about our augmented reality, the demo will including the whole process, including locating the QR code, find the rendering plane and our 3D model on it.

What we hope to deliver?

Augmented reality is, at its core, an interactive medium, giving the user an immersive visual experience in real time. We wish to produce an interactive demo wherein, but this remains a stretch goal due to the immense challenges it entails. We would have to develop a mobile application for our project and optimize rendering very heavily to allow for high frame frates (all the while using less powerful hardware!)

Further than 3D model, we want to try something new. For example, some animation as shown in the youtube vedio.Or making it more genral, not limited to the QR code, so that we can do something like a real time shoes wearing for advertisement.

I thought the most important and difficult part is how to combine the computer graphics rendering skill with the computer vision together. More specifically, that is about how to recognize the QR code first, then use the recognition result to convey the position and height message with the tilt angle for the following rendering part. There is no qualitative analysis for our project so far, maybe we will consider the rendering speed later. At this time, We thought that the most effective and convenient way is to watch the vedio and check the quality of the image or 3D model by eye bowls. It's easy to tell whether that is cool or not.

1.the AR position is stable, how to add stability? 2. How we gonna do the render? About the combination of these 1. Detect QR code 2. Overlay 2d 3. Overlay 3d 4. More complex 3d 5. Rendering speedup / Complex rendering
